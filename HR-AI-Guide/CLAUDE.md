## ✅ 합의 완료된 설계 결정 (변경 금지, 재논의 대상 아님)

- 가이드 위치: 전사 8대 수칙의 HR 특화 시행기준
- 핵심 독자: "어쩌다 기밀급 데이터를 마주치는" 일반 HR 담당자
- 핵심 메시지: "쓰지 말라가 아니라 안전하게 쓰시라"
- 산출물 형태: 검색형 HTML (주 산출물) + 데이터 마스터 엑셀 (Lab 검토용)
- 톤: 정책문서 어미 + 활성화 메시지
- 법규 범위: 한국법인 기준 (개보법 + 근기법 + 자본시장법)
- 데이터 분류: J 회사 10대 분류 그대로 사용, 가이드는 변경 안 함
- 5대 원소 명명: A/B/C/D/E + 한국어 병기
- Level 1/2/3 정의: 원소 기반으로 명확화
  - Level 1 = A만 처리
  - Level 2 = A+B+C 처리
  - Level 3 = A+B+C+D 처리, E는 원칙 제거
- 건강 정보: RED 준용 별도 명시
- 문의 채널명: AX&CI Lab

## ❌ 반복하면 안 되는 권고 (요약)

자세한 내용은 `docs/05_철회된_권고.md` 참조. 핵심은:

- J 회사 정보보호센터가 이미 결정한 분류는 **그대로 존중**
- M&A·평판·핵심인재 등을 🔴로 옮기자는 권고는 모두 **철회됨**
- 회사가 "기밀 최소화 + 권한관리 단위 최소화" 전략을 채택했음을 잊지 말 것
- LLM의 역할: 분류 *변경*이 아니라 분류에 따른 *행동 가이드*
