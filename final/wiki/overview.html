Overview

사내망에서 누구나 1분 안에 띄울 수 있는 HR AX 컨셉 작업대. ① 동작하는 MVP를 직접 만져보게 하고, ② 아직 합의되지 않은 정의·개념·시나리오를 같이 빚는다. 합의가 끝난 항목은 컨플루언스로, 결정된 작업은 지라로 빠져나간다.

1. 한 줄 정의

HR AX Platform은 HR 특성을 고려한 Data Governance와 Harness Engineering을 기반으로, 민감한 HR 데이터와 업무 맥락을 안전하게 통제하며 사내외 LLM을 HR 업무 실행에 연결하는 People팀 단일 책임 독립 플랫폼.

3. 사용자 시나리오

Open Chat Mode

채팅으로 먼저 시작

사용자가 자연어로 요청하면 플랫폼이 로그인 권한, 사업부, 도메인, 민감도, L3-L6 후보를 뒤에서 추정하고 필요한 경우 추천 업무 흐름으로 전환합니다. 사용자가 도메인이나 L6를 직접 고르지 않아도 됩니다.

Guided Work Mode

추천 업무로 시작

사용자의 조직·역할·최근 업무 패턴을 기준으로 Work Card를 제안하고, 선택 즉시 Agent · Skill · Tool · Context를 자동 조립합니다. L6는 사용자가 찾는 대상이 아니라 플랫폼이 붙이는 메타데이터입니다.

두 모드는 경쟁 관계가 아니라 같은 통제 엔진을 공유하는 진입 방식입니다. 어느 쪽으로 시작해도 권한 확인, 민감도 분류, 비식별화, 모델 라우팅, 감사 로그, 검수형 자산화는 동일하게 적용됩니다.
