HR AX Platform

HR 특성을 고려한 Data Governance와 Harness Engineering을 기반으로, 민감한 HR 데이터와 업무 맥락을 안전하게 통제하며 사내외 LLM을 HR 업무 실행에 연결하는 독립 플랫폼.

한 줄 정의

HR AX Platform은 HR 데이터의 보안·권한·자산화를 통제하는 Data Governance와, LLM이 HR 맥락 위에서 일하도록 만드는 Harness Engineering을 결합한 People팀 단일 책임 독립 플랫폼.

두 가지 진입 모드

Open Chat Mode

채팅으로 먼저 시작

사용자가 도메인이나 L6를 직접 고르지 않아도 된다. 플랫폼이 입력 의도, 권한 범위, 민감도, L3-L6 후보를 뒤에서 추정한다.

Guided Work Mode

추천 업무로 시작

로그인 권한과 반복 업무 패턴을 기준으로 Work Card를 제안한다. L6는 사용자가 찾는 대상이 아니라 플랫폼이 붙이는 메타데이터다.
